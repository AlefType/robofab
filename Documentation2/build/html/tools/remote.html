remote¶

Remote control for MacOS FontLab. initFontLabRemote() registers a callback for appleevents and runFontLabRemote() sends the code from a different application, such as a Mac Python IDE or Python interpreter.

robofab.tools.remote.initFontLabRemote()[source]¶

Call this in FontLab at startup of the application to switch on the remote.

robofab.tools.remote.runFontLabRemote(code)[source]¶

Call this in the MacOS Python IDE to make FontLab execute the code.
