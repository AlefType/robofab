adapterPens¶

class robofab.pens.adapterPens.FabToFontToolsPenAdapter(fontToolsPen)[source]¶

Class that covers up the subtle differences between RoboFab Pens and FontTools Pens. ‘Fab should eventually move to FontTools Pens, this class may help to make the transition smoother.

class robofab.pens.adapterPens.GuessSmoothPointPen(outPen)[source]¶

Filtering PointPen that tries to determine whether an on-curve point should be “smooth”, ie. that it’s a “tangent” point or a “curve” point.

class robofab.pens.adapterPens.PointToSegmentPen(segmentPen, outputImpliedClosingLine=False)[source]¶

class robofab.pens.adapterPens.SegmentToPointPen(pointPen, guessSmooth=True)[source]¶

class robofab.pens.adapterPens.TransformPointPen(outPen, transformation)[source]¶
